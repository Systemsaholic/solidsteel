Homepage Video Test

Instructions:

  1. Open browser developer tools (F12)
  2. Go to Console tab
  3. Visit the homepage: Homepage
  4. Check for any video-related errors in console
  5. Check Network tab to see if video is loading

What you should see:

  • A background video playing automatically in the hero section
  • Video should be muted and looping
  • If video fails, it should fall back to the static image
  • In development mode, you'll see a status indicator in top-right corner

Debug in Console:

// Check if video element exists
console.log('Video element:', document.querySelector('video'));

// Check video properties
const video = document.querySelector('video');
if (video) {
    console.log('Video source:', video.currentSrc);
    console.log('Video ready state:', video.readyState);
    console.log('Video network state:', video.networkState);
    console.log('Video error:', video.error);
}
