Gallery Modal Test Instructions

  1. Open browser developer tools (F12)
  2. Go to Console tab
  3. Visit the project page: Greystone Village Project
  4. Wait for page to load (you should see project content, not "Loading...")
  5. Look for any console errors
  6. Try clicking on the main hero image
  7. Try clicking on gallery images below

What should happen:

  • Images should show an expand icon on hover
  • Clicking an image should open a fullscreen modal
  • Modal should have previous/next navigation
  • ESC key or clicking outside should close modal

Debugging Steps:

In browser console, run:

// Check if React is loaded
console.log('React loaded:', typeof React !== 'undefined');

// Check for gallery elements
console.log('Gallery images found:', document.querySelectorAll('.cursor-pointer.group').length);

// Check for modal in DOM
console.log('Dialog elements:', document.querySelectorAll('[role="dialog"]').length);
